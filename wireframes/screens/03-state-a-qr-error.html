  • QR code was scanned but is unreadable or points to an invalid / unrecognised gateway.

Design notes
  • Corner colour changes from white → #f97066 (error red) — no other UI change needed.
  • Scan line animation is stopped in the error state — scanning has halted.
  • Error banner uses rgba(240,68,56,.15) background with #f97066 border — dark-mode friendly.
  • Camera feed is still visible in background — user can re-aim and retry (button not shown here; implemented as auto-retry after dismiss).
